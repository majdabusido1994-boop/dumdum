# Images Folder

Drop DUMDUM's images here. The website expects these files:

## Required Images
- `hero-poster.jpg` — Hero section fallback (when video isn't loaded). Wide landscape, moody/dark. (1920x1080)
- `about.jpg` — Portrait photo for the About section. (800x1200, 3:4 ratio)
- `og-image.jpg` — Social media share image. (1200x630)

## Tips
- Use dark, moody, high-contrast photos
- Compress images (use TinyPNG or Squoosh)
- JPG for photos, PNG only if transparency is needed
