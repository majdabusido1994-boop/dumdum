# Videos Folder

Drop DUMDUM's videos here. The website expects:

## Required Videos
- `hero-bg.mp4` — Background video for the hero section. Should be:
  - 15-30 seconds long, looping
  - Moody/dark footage (mixing, crowd, Amman night scenes)
  - 1080p resolution max (keep file size under 10MB for fast loading)
  - No audio needed (plays muted)

## Tips
- Compress with HandBrake or FFmpeg: `ffmpeg -i input.mp4 -vcodec h264 -acodec aac -crf 28 hero-bg.mp4`
- Keep it dark and atmospheric — it plays behind text at 30% opacity
- Short clips of: turntable close-ups, crowd silhouettes, city lights, neon reflections
